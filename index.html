Segunda entrega - JavaScript

La aplicación consta de una Calculadora de Pintura para el proyecto de Desarrollo Web de una pintureria. En la misma hice uso de un constructor de objetos que vendrian siendo los ambientes, que constan de "nombre del ambiente", "Cantidad de Metros", "Color" y finalmente "Cantidad de manos de pintura". El cálculo es metros cuadrados x cantidad de manos x 0.11 lts. La aplicacion muestra los litros de pintura de cada ambiente y finalmente muestra el total de litros de pintura de cada color para toda la casa.
